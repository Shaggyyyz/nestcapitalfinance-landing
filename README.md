# nestcapitalfinance-landing
nestcapitalfinance-landing

A simple, single-page landing website for Nest Capital Finance, built with plain HTML and styled using Tailwind CSS (via CDN).

This repository contains the source code for a responsive, multi-view landing page designed to be visually clean and highly aesthetic with a light gray color scheme (#F5F5F5).

The entire website is contained within a single index.html file, making it extremely easy to deploy.

GitHub Pages

The fastest way to deploy this site is using GitHub Pages:

Commit: Ensure the index.html file is in the root directory of this repository.

Settings: Go to your repository Settings tab on GitHub.

Pages: Navigate to the Pages section in the left sidebar.

Source: Under "Build and deployment," set the source to Deploy from a branch and select the main (or master) branch and the / (root) folder.

Save: Save your changes. GitHub will build and deploy the site, usually within a few minutes. The live URL will be provided on the Pages settings screen.

The current aesthetic is a Light Gray Theme based on the following custom colors defined in the <style> block:

Accent/Text Color: #333333 (Dark Grey)

The responsiveness and styling are handled entirely by utility classes from the Tailwind CSS CDN.
